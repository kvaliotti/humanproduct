# The 9-Question Positioning Pressure Test

Canonical question definitions, shared by `positioning-workshop` (quick application during finalization), `positioning-review` (full scoring rubric), and `positioning-orchestrator` (Phase 4 capture). Every "no" or "red flag" requires a fix before finalizing.

| # | Test | Core Question | Pass Criteria | Red Flag |
|---|------|----------------|--------------|----------|
| 1 | **Substitution test** | If a competitor swapped their name into your positioning narrative, would it still sound true? | It would sound FALSE — claims are specific enough that only this product fits | If it still sounds true, the positioning is generic — you haven't captured what's actually unique |
| 2 | **10-names test** | Can you name 10 real customers or accounts that fit the target market definition? | Yes, immediately | If you can't, the ICP is either too broad (can't identify specific matches) or too narrow (not enough exist) |
| 3 | **Alternative test** | Do competitive alternatives include non-obvious options (spreadsheets, humans, doing nothing)? | Yes — direct competitors AND at least 2 non-obvious options | If only direct competitors are listed, you're missing the real competition your prospects face |
| 4 | **Adjective test** | How many hype adjectives appear (seamless, powerful, intuitive, best-in-class, cutting-edge)? | Zero | Every hype adjective is a red flag — it means you haven't found the specific, provable claim underneath |
| 5 | **Proof test** | Does every value theme cite specific, checkable proof (data, customer quotes, third-party reviews)? | Yes, for every theme | Any theme without proof should be demoted to "hypothesis" or cut entirely |
| 6 | **Category test** | Does the chosen market category help prospects correctly predict what features to expect? | Yes | If an outsider would expect a different feature set from the category you picked, it's wrong |
| 7 | **Kill-the-tagline test** | If you deleted the tagline and only kept the 5+1 canvas, could a salesperson still pitch? | Yes — the canvas contains all the substance | If the tagline is doing all the work, the underlying positioning is weak |
| 8 | **Sharpness test** | Can a new hire explain the positioning in 60 seconds after reading it once? | Yes | If it requires context, history, or caveats to explain, it's not sharp enough |
| 9 | **Cross-functional alignment test** | Would sales, marketing, product, and CS all describe the product the same way using this positioning? | Yes | If each team tells a different story, positioning hasn't been properly cascaded |

## Most Common Failures

Most positioning fails tests 1 (substitution), 4 (adjectives), and 7 (tagline dependency). These are the most common and easiest to fix.
