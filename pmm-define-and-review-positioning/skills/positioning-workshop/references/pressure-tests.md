# Positioning Pressure Tests

Run each test against draft positioning. Every "no" or "red flag" requires a fix before finalizing.

## The 9-Question Stress Test

| # | Test | Pass Criteria | Red Flag |
|---|------|--------------|----------|
| 1 | **Substitution test** | If a competitor swapped their name into your positioning narrative, it would sound FALSE | If it still sounds true, the positioning is generic — you haven't captured what's actually unique |
| 2 | **10-names test** | You can name 10 real customers or accounts that fit the target market definition | If you can't, the ICP is either too broad (can't identify specific matches) or too narrow (not enough exist) |
| 3 | **Alternative test** | Competitive alternatives include non-obvious options (spreadsheets, humans, doing nothing) | If only direct competitors are listed, you're missing the real competition your prospects face |
| 4 | **Adjective test** | Count hype adjectives: seamless, powerful, intuitive, best-in-class, cutting-edge. Target: ZERO | Every hype adjective is a red flag — it means you haven't found the specific, provable claim underneath |
| 5 | **Proof test** | Every value theme cites specific, checkable proof (data, customer quotes, third-party reviews) | Any theme without proof should be demoted to "hypothesis" or cut entirely |
| 6 | **Category test** | The chosen market category helps prospects correctly predict what features to expect | If an outsider would expect a different feature set from the category you picked, it's wrong |
| 7 | **Kill-the-tagline test** | If you deleted the tagline and only kept the 5+1 canvas, could a salesperson still pitch? | If the tagline is doing all the work, the underlying positioning is weak |
| 8 | **Sharpness test** | A new hire can explain the positioning in 60 seconds after reading it once | If it requires context, history, or caveats to explain, it's not sharp enough |
| 9 | **Cross-functional alignment test** | Sales, marketing, product, and CS would all describe the product the same way using this positioning | If each team tells a different story, positioning hasn't been properly cascaded |

## How to Use

After completing the positioning canvas and narrative:

1. Run each test in order
2. For each failure, diagnose which positioning component is weak
3. Loop back to the relevant step (usually Step 4, 5, or 8)
4. Re-run failed tests after patching

Most positioning fails tests 1 (substitution), 4 (adjectives), and 7 (tagline dependency). These are the most common and easiest to fix.
