# Positioning Pressure Tests (Workshop Quick Reference)

Run each test against draft positioning. Every "no" or "red flag" requires a fix before finalizing.

For the full definition of each of the 9 questions (pass criteria + red flags), see the shared reference: `${CLAUDE_PLUGIN_ROOT}/references/pressure-test.md`. For the detailed 3-tier scoring rubric with fix directions (used during a formal audit), see `${CLAUDE_PLUGIN_ROOT}/skills/positioning-review/references/pressure-test-scoring.md`.

## The 9 Tests (names only — see shared reference for detail)

1. Substitution test
2. 10-names test
3. Alternative test
4. Adjective test
5. Proof test
6. Category test
7. Kill-the-tagline test
8. Sharpness test
9. Cross-functional alignment test

## How to Use (Workshop Context)

After completing the positioning canvas and narrative:

1. Run each test in order (pull questions and criteria from `${CLAUDE_PLUGIN_ROOT}/references/pressure-test.md`)
2. For each failure, diagnose which positioning component is weak
3. Loop back to the relevant step (usually Step 4, 5, or 8)
4. Re-run failed tests after patching

Most positioning fails tests 1 (substitution), 4 (adjectives), and 7 (tagline dependency). These are the most common and easiest to fix.
