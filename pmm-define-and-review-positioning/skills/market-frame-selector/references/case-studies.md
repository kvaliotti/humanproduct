# Market Frame Case Studies

## Head to Head: Database → Data Warehouse
- Positioned as "database" → couldn't compete with Oracle
- Moved to "data warehouse" — existing category, no clear leader
- Strengths became obvious in new frame
- **Lesson:** Head to Head works when you enter a category where you can win

## Big Fish, Small Pond: Janna Systems
- Couldn't beat Siebel in general CRM
- Carved "CRM for investment banks"
- $2M → $70M in 18 months, acquired for $1.7B
- **Lesson:** Same product, narrower frame = decisive advantage

## Create New Game: Eloqua
- Coined "marketing automation"
- $12M → $96M over 6 years, acquired for $870M
- **Lesson:** Category creation works but demands massive investment and patience

## Reframing: Clearpath → OTTO Motors
- "Robots" triggered wrong assumptions
- "Self-driving vehicles for industrial uses" triggered right ones
- **Lesson:** Category name shapes feature expectations. Wrong name = wrong fight.

## Trend Layering: Redgate + DevOps
- Led database tools but lacked urgency
- Layered "database DevOps" without changing core positioning
- 100% increase in inbound leads
- **Lesson:** Trends amplify existing positioning. They don't replace it.
