# Pressure Test Scoring Rubric

## Test 1: Substitution Test

**Question:** If a competitor swapped their product name into your positioning narrative, would it still sound true?

| Score | Criteria |
|-------|----------|
| ✅ Pass | Positioning contains specific, provable claims that only apply to this product. A competitor insertion would sound false. |
| ⚠️ Weak | Most of the positioning is specific, but 1-2 phrases are generic enough to apply to competitors (e.g., "easy to use", "saves time") |
| ❌ Fail | A competitor could use most or all of this positioning unchanged. Nothing distinguishes this product. |

**Fix direction:** Return to Step 5 (unique attributes). The attributes listed aren't truly unique or aren't being translated into the positioning.

## Test 2: 10-Names Test

**Question:** Can you name 10 real customers or target accounts that fit the target market definition?

| Score | Criteria |
|-------|----------|
| ✅ Pass | User can immediately name 10+ accounts. Target definition is specific and actionable. |
| ⚠️ Weak | User can name 3-9 accounts but has to stretch. Definition has some ambiguity. |
| ❌ Fail | User can't name specific accounts. Target is too broad ("any company with data") or too abstract. |

**Fix direction:** Return to Step 7 (who cares). Narrow the characteristics until a prospect list can be built.

## Test 3: Alternative Test

**Question:** Are competitive alternatives comprehensive — including non-obvious options?

| Score | Criteria |
|-------|----------|
| ✅ Pass | Alternatives include direct competitors AND at least 2 non-obvious options (manual processes, general-purpose tools, status quo, combinations). |
| ⚠️ Weak | Alternatives include some non-obvious options but miss important ones. |
| ❌ Fail | Only direct competitors listed. Missing "do nothing", manual workarounds, spreadsheets, or combination approaches. |

**Fix direction:** Return to Step 4. Ask best-fit customers directly: "What would you use if we didn't exist?"

## Test 4: Adjective Test

**Question:** How many hype adjectives appear in the positioning?

| Score | Criteria |
|-------|----------|
| ✅ Pass | Zero hype adjectives. Every claim is specific and factual. |
| ⚠️ Weak | 1-2 hype adjectives that could be replaced with specifics. |
| ❌ Fail | 3+ hype adjectives (seamless, powerful, intuitive, cutting-edge, best-in-class, world-class, robust, comprehensive, innovative). |

**Banned words:** seamless, powerful, intuitive, cutting-edge, best-in-class, world-class, robust, comprehensive, innovative, next-generation, revolutionary, disruptive, game-changing, holistic, synergistic, scalable (unless with specific metrics).

**Fix direction:** For each adjective, ask "What specifically makes this true?" Replace the adjective with the specific answer. "Powerful analytics" → "Sub-second queries across 100M+ rows."

## Test 5: Proof Test

**Question:** Does every value theme cite specific, checkable proof?

| Score | Criteria |
|-------|----------|
| ✅ Pass | Every value theme has proof: customer quotes, case study metrics, third-party reviews, measurable outcomes. |
| ⚠️ Weak | Some themes have proof; 1-2 rely on claims without evidence. |
| ❌ Fail | Value themes are stated as claims. No proof, no metrics, no third-party validation. |

**Acceptable proof:** Published case study, named customer quote, third-party review/award, specific measurable outcome (e.g., "$50K/year savings"), comparison data.

**Fix direction:** For each unproven theme, either find proof or demote the theme. A value claim without proof is marketing fiction.

## Test 6: Category Test

**Question:** Does the market category trigger correct feature expectations?

| Score | Criteria |
|-------|----------|
| ✅ Pass | An informed outsider hearing the category would correctly predict the product's core features and competitive set. |
| ⚠️ Weak | Category is approximately right but triggers some wrong expectations (prospects ask about features you don't have). |
| ❌ Fail | Category triggers fundamentally wrong assumptions. Prospects expect a different product than what you deliver. |

**Fix direction:** Return to Step 8. The market frame isn't working. Consider Big Fish Small Pond (add a subsegment qualifier) or an adjacent category.

## Test 7: Kill-the-Tagline Test

**Question:** If you deleted the tagline and only had the 5+1 canvas, could a salesperson still pitch?

| Score | Criteria |
|-------|----------|
| ✅ Pass | The canvas contains all the substance. Tagline is optional decoration. |
| ⚠️ Weak | Canvas is mostly complete but missing 1-2 elements that the tagline is compensating for. |
| ❌ Fail | Tagline is doing all the work. Canvas is empty or generic. |

**Fix direction:** The canvas IS the positioning. If it can't stand alone, the positioning isn't done.

## Test 8: Sharpness Test

**Question:** Can a new hire explain the positioning in 60 seconds after reading it once?

| Score | Criteria |
|-------|----------|
| ✅ Pass | Positioning is clear enough for immediate comprehension. No jargon, no caveats needed. |
| ⚠️ Weak | Mostly clear but requires 1-2 clarifications or domain knowledge. |
| ❌ Fail | Requires significant context, insider knowledge, or history to explain. |

**Fix direction:** Simplify. If it takes more than 60 seconds, it's not positioning — it's a strategy document. Strip to essentials.

## Test 9: Cross-Functional Alignment Test

**Question:** Would sales, marketing, product, and CS all tell the same story?

| Score | Criteria |
|-------|----------|
| ✅ Pass | Positioning is documented and shared. All teams describe the product consistently. |
| ⚠️ Weak | Core story is consistent but different teams emphasize different aspects. |
| ❌ Fail | Each team tells a different story. No shared positioning document exists. |

**Fix direction:** This is an execution problem, not a positioning problem. Create the canvas, build the sales story, create a master messaging document. Cascade across teams.
