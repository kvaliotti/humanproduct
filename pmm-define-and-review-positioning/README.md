# PMM: Define and Review Positioning

A structured product positioning toolkit. Define positioning through a guided 10-step workshop, audit existing positioning, map competitive alternatives, select market frames, and build sales stories.

## Skills

| Skill | What It Does | Trigger Phrases |
|-------|-------------|-----------------|
| **positioning-workshop** | Full guided 10-step positioning process | "position my product", "positioning workshop" |
| **positioning-review** | Audit and stress-test existing positioning | "review my positioning", "audit positioning", "stress-test positioning" |
| **competitive-alternatives** | Deep dive: alternatives → attributes → value themes | "map competitive alternatives", "what would customers use instead" |
| **market-frame-selector** | Choose between Head to Head / Big Fish Small Pond / New Category | "what market should we be in", "positioning style" |
| **sales-story-builder** | Build 7-stage sales narrative from completed positioning | "build sales story", "translate positioning to sales" |
| **positioning-orchestrator** | End-to-end pipeline: all skills in sequence | "full positioning process", "end-to-end positioning" |

## Usage

**Starting from scratch:** Use `positioning-orchestrator` or `positioning-workshop` for the full guided experience.

**Reviewing existing work:** Use `positioning-review` with your current positioning document, canvas, or homepage.

**Focused work on specific steps:** Use `competitive-alternatives` or `market-frame-selector` independently when you need depth on a specific positioning component.

**After positioning is done:** Use `sales-story-builder` to translate the canvas into a sales narrative.

## Framework Summary

The 5+1 positioning components (in dependency order):

1. **Competitive Alternatives** — what customers would use without you
2. **Unique Attributes** — what you have that alternatives lack
3. **Value Themes** — why someone would care, with proof
4. **Target Market** — who cares the most
5. **Market Category** — the frame that makes value obvious
6. **(Optional) Trends** — why it matters right now
